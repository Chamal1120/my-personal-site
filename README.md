- Pnpm

## Build Instructions

#### Prerequisites

- Node v20 or above
- pnpm latest

#### Steps

1. Fork and Clone the repo:

```bash
git clone "your-git-url"
```

2. Change into the cloned repo:

```bash
cd "your-fork's-name"
```

3. Install deps:

```bash
pnpm i
```

4. Run dev server:

```bash
pnpm run dev
```

5. Do your adjustments.

6. Build the production build:

```bash
pnpm run build
```

7. Deploy to your choice of PaaS or any other infra.
